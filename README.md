# Operating Systems Exercises (C / Linux)
This repository contains 5 programming assignments from the Operating Systems course, implemented in C in a Linux environment.
The exercises focus on low-level programming and operating system concepts such as process management, synchronization, and inter-process communication.

## Topics Covered
- Process creation using `fork()`
- Shared memory (System V IPC)
- Signal handling
- POSIX threads (pthreads)
- Semaphores (POSIX)
- Critical section problems

## Technologies
- C
- GCC
- Linux
- POSIX Threads
- System V IPC
- POSIX Semaphores

## Purpose
Academic exercises demonstrating practical understanding of operating system concepts.
